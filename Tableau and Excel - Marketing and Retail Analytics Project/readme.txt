This directory consists of Analysis done on a retail e-commerce website 'OList'
It consist of:
  - Excel file of raw data that we got from the company
  - Clean version of Excel file done in Excel using formulas like VLOOKUP, COUNTBLANK, etc and graphs also created in Excel to detect outliers
  - A Word document that explains in detail all that is done to clean the dataset
  - Tableau dashboard with Customer Analysis, Product Analysis, Market Basket Analysis and Pareto charts based on quantity and revenue
  - A PPT that summarizes all the observations made by doing the analysis and some recommendations to the company
